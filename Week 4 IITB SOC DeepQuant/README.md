# American Put Option Pricing using the CRR Binomial Tree Model

This project implements the Cox Ross Rubinstein (CRR) Binomial Tree model for pricing European and American Put Options in Python. The implementation also includes a convergence analysis, an early exercise boundary visualization, and a 3D option price surface. To run the project, make sure Python 3 and the required libraries (`numpy` and `matplotlib`) are installed. Keep `american_puts.py`, `test_american_put.py`, and `Week 4 Report.ipynb` in the same folder. Open the notebook and run all the cells from top to bottom to reproduce the pricing results, convergence table, and plots. Alternatively, you can execute `test_american_put.py` from the terminal to verify the implementation and generate the convergence results.
